【1. 现状重构与起笔】
上一幕最后已发生：苏瑾迟到15分钟后推门进入排练室。
已完成动作：所有人抬头，但没人说话。
尚未完成动作：苏瑾不知道刚才发生了什么，需要判断气氛并加入排练。
关键人物当前位置：老周在鼓组后调音，小唐靠墙刷手机，阿杰坐在音箱上翻谱子，苏瑾刚推开门站在门口。
当前可见物件：鼓组、贝斯靠在墙上、阿杰手里的谱子、小唐的手机、苏瑾的吉他盒。
当前具体问题：苏瑾需要在不被直接告知的情况下，判断乐队对自己的态度，并顺利开始排练。
正文最合适的第一画面：苏瑾的手还握在门把上，门在身后合拢，她面对三个沉默的队友。

【2. 信息边界】
苏瑾：
- 已知：自己迟到15分钟；成员们刚才抬头看了她但没说话；老周在调音、小唐刷手机、阿杰翻谱子。
- 未知：成员们刚才聊了什么；每个人对自己迟到的真实态度；阿杰为什么背对着她翻谱子却不翻页。
- 可靠来源：眼前可见的行为、物件状态、接下来的互动反馈。
- 需要证据：成员是否愿意配合排练；阿杰是否在等她解释。
- 最容易被越权公布：苏瑾的迟到原因、成员们的内心评价。

老周/阿杰/小唐的后台信息只通过行为表达，旁白不得直接说出。

【3. 人物活化】
苏瑾：
- 核心性格：敏感、自尊心强、习惯用行动而非言语解决问题。
- 当前压力：迟到带来的愧疚感和被排斥的恐惧。
- 本场被激活的人格侧面：观察者——她会先收集信息再行动，但信息不足时会选择用音乐试探。
- 最刻板反应：直接道歉、解释迟到原因。
- 更具体的方向：不道歉，先通过物件和动作判断每个人的态度，然后用吉他提出一个音乐上的问题。
- 对白习惯：话少，用行动代替解释；提问时直接、简短。

阿杰：
- 最刻板反应：冷着脸质问“怎么迟到了”。
- 更具体的方向：用翻谱子掩饰烦躁，不主动开口，但会在音乐配合中暴露真实情绪。

老周：
- 最刻板反应：打圆场说“来了就好”。
- 更具体的方向：用调音的断续节奏表达不安，用鼓点回应苏瑾的和弦，但不主动调解。

小唐：
- 最刻板反应：抬头看一眼继续刷手机。
- 更具体的方向：刷手机的动作有节奏感，可能在偷偷听现场动静，用漠不关心保护自己不被卷入。

【4. 人物套路淘汰】
作废1：苏瑾一进门就道歉，解释迟到原因。
→ 理由：这会让冲突立刻消解，失去观察和试探的过程。苏瑾的性格是用行动弥补，不是用言语辩解。

作废2：老周主动打圆场，缓解气氛。
→ 理由：老周的中立不是和事佬，他用音乐而非言语处理问题。让他主动调解会削弱乐队内部的真实张力。

【5. 剧情套路淘汰】
作废1：阿杰直接质问苏瑾为什么迟到，两人发生口角。
→ 理由：用对话解决矛盾太直接，缺乏音乐场景特有的非语言对抗。冲突应该通过排练中的配合与不配合来表达。

作废2：苏瑾弹完第一个和弦后，所有人立刻跟上，排练顺利开始。
→ 理由：让问题被一个和弦解决是廉价的。成员的反应应该有延迟、有保留，配合中的生硬才是真实状态。

选择路线：苏瑾通过观察→调弦试探→第一个和弦提出音乐问题→成员以各自的方式回应（老周跟上但有停顿，小唐放下手机但慢了半拍，阿杰转身但歌声延迟加入）→可见的生硬配合揭示矛盾未消。

【6. 句式套路淘汰】
作废1：“她能感觉到空气里的紧张。” → 直接心理解释，改为写苏瑾观察到的具体事实。
作废2：“阿杰显然在生气。” → 他人动机断言，改为写阿杰翻谱子没翻页、背对苏瑾的具体动作。
作废3：“排练室里的沉默像一堵墙。” → 俗套比喻，改为写门合拢后的声音、空调的低频噪音、老周调音时断时续的鼓点。

【7. 行动与篇幅骨架】
因果链：
苏瑾推门进入，门在身后合拢
→ 苏瑾扫视三人行为，判断无人主动开口
→ 苏瑾决定不道歉，走向自己的位置，打开吉他盒
→ 老周继续调音但节奏断续，小唐刷手机没抬头，阿杰翻谱子没翻页
→ 苏瑾调弦，用琴弦的声音试探现场
→ 无人响应，苏瑾确认气氛异常
→ 苏瑾弹奏第一个和弦，提出音乐上的问题
→ 老周用鼓点回应，但节奏略微迟疑；小唐放下手机拿起贝斯，动作慢了半拍；阿杰缓缓转身，仍没看苏瑾
→ 阿杰的歌声延迟两拍加入，配合明显生硬
→ 排练开始，但每个人都保留了距离
→ stop_state成立

【8. 停止与文风保护】
第一个满足stop_state的可见事实：阿杰的歌声延迟加入，老周的鼓点略显生硬，排练开始但配合明显不协调。
最容易追加的两种多余内容：苏瑾内心总结“他们还在生我的气”；环境收尾“窗外的阳光照进来”。
叙事距离：紧密跟随苏瑾的感知，但不进入内心独白式解释。
对白风格：极少对白，以音乐和动作为主要交流方式。苏瑾如有对白，简短直接。
最易出现的OOC措辞：苏瑾用书面语思考（“我能感知到团队凝聚力的下降”），阿杰突然用温柔语气说话。

stop_state成立后作废：苏瑾再次确认气氛、老周用眼神安慰苏瑾、排练室外的环境描写收尾。
