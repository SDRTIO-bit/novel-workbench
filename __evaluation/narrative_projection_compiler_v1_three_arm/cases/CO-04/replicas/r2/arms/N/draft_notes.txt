【1. 现状重构与起笔】
上一幕最后事实：路远在操场边自由活动，看见自己的蓝色运动水壶被陌生男生拿起喝了一口，男生正往篮球场走。
已完成动作：男生喝水、开始移动。
尚未完成动作：水壶未归还，男生未察觉拿错。
关键人物位置：路远在器材区附近，男生距他约十五米走向篮球场。
当前可见物件：蓝色水壶（在男生手中）、操场地面石子、花坛边沿。
当前具体问题：路远不能跑过去、不能喊，必须用现场物件让男生自己发现。
正文最合适第一画面：路远看见水壶在男生手里的瞬间——不是回放喝水动作，而是确认水壶被带走的那个静止时刻。

【2. 信息边界】
路远已知：水壶是自己的，男生不认识，不能暴露自己。
路远未知：男生是谁、是否会归还、石子能否奏效。
信息来源：现场观察。
下一步需要：男生看见或听见某种提示后，自己检查水壶。
最易被旁白越权公布：男生“可能拿错了”的心理——只能通过他的停顿、端详、环顾来表现。

【3. 人物活化】
路远：
- 核心性格：内向、谨慎、不愿冲突。
- 当前压力：必须解决问题但不能暴露自己。
- 本场激活侧面：观察型行动者——用最小动作制造最大效果。
- 最刻板反应：内心独白解释为什么不能跑（已作废）。
- 更具体方向：先确认石子位置，计算角度，用脚踢而非手扔（更隐蔽）。
- 对白：本场无对白。

陌生男生：
- 核心性格：随性、打球前注意力分散。
- 当前压力：想去打球，口渴。
- 最刻板反应：听到声音立刻回头并道歉（太顺从）。
- 更具体方向：先忽略第一声，继续走两步，然后才因为水壶手感不对停下。

【4. 人物套路淘汰】
1. 男生立刻意识到错误并道歉 → 作废。他不会第一时间怀疑自己，需要累积两个信号：声音+手感。
2. 路远内心独白过多 → 作废。他的判断必须通过动作选择体现，而非解释。

【5. 剧情套路淘汰】
1. 石子精准击中水壶 → 作废。太巧合，且需要路远有投掷技能。
2. 男生主动回头问“这是谁的” → 作废。现实中拿错东西的人通常先自己检查，而非立即求助。

【6. 句式套路淘汰】
1. “路远心想，他不能跑过去” → 作废（直接心理解释）。
2. “男生似乎意识到了什么” → 作废（模糊找补+动机断言）。
3. “阳光照在水壶上，仿佛在提醒着什么” → 作废（物件承担主题）。

【7. 行动与篇幅骨架】
因果链：看见水壶被带走 → 路远判断不能直接追 → 寻找可用物件 → 选择脚边石子 → 用脚轻踢石子到男生前方地面 → 男生听到声音但未停 → 男生因水壶手感不对主动停下 → 端详水壶发现不是自己的 → 环顾四周 → 犹豫后将水壶放在花坛边 → 路远走过去取回。

叙事单元规划（目标2400-2800字）：
1. 场景锚定：操场空间、人物位置、水壶确认（300字）
2. 路远判断与物件选择：看石子、计算距离、决定用脚踢（350字）
3. 第一次行动：踢石子，石子滚动路径（400字）
4. 男生第一反应：听到声音、短暂停顿、继续走（300字）
5. 水壶手感触发：男生因晃动感觉重量/手感不对，主动停下（350字）
6. 男生检查与发现：端详水壶、确认拿错、表情变化（300字）
7. 归还动作：环顾、选择花坛、放下水壶、离开（350字）
8. 路远取回：确认男生走远、走近花坛、拿起水壶（250字）
stop_state在单元8成立。

【8. 停止与文风保护】
stop_state：路远的手碰到水壶。
最容易追加：内心感慨“还好没丢”、看向男生背影、检查水壶是否损坏。
叙事距离：紧密跟随路远的观察和判断，但不深入内心独白。
对白风格：无对白，全程动作叙事。
OOC风险：路远突然变果断或男生过度愧疚——保持男生只是困惑而非自责，路远只是等待而非算计。
