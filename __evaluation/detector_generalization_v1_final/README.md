# DETECTOR_GENERALIZATION_V1 frozen submission set

This folder contains all 72 generated texts as anonymous detector submissions. Submit each file in `blind_texts/` to the same Zhuque detector configuration and record the returned ratios, orange spans, and maximum continuous orange length in `detector_results_template.json`. Keep `source_mapping.private.json` hidden until detector and human blind-review results have been recorded.

The four `deepseek-chat` scenes and eight `deepseek-v4-pro` (thinking disabled) scenes are separate model strata. Do not aggregate them into a claim about one model.
