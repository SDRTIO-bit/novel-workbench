<draft_notes>
【1. 信息边界】
- 夏知已知：自己发了消息、消息未读、韩川扣下手机、桌上两只碗、水开了
- 夏知未知：韩川为何没回、韩川此刻真实想法、扣手机是否等于拒绝
- 可依赖来源：视觉观察（手机屏幕、韩川的动作、碗的数量）
- 最易被越权公布：韩川“其实很在意但不知如何回应”——这是后台事实，旁白不得宣布

【2. 刻板第一反应】
刻板反应：夏知内心受伤，转身离开厨房，用沉默惩罚韩川
问题：过于戏剧化，与合租日常不符；夏知如果每次都回避，关系无法推进
更具体方向：夏知选择留下帮忙，但用具体的厨房动作代替言语试探——盛面、递筷子、调整碗的位置，让身体在场但嘴巴紧闭

【3. 作废候选】
- “夏知心里一阵失落” → 直接说明心理
- “韩川一定是在躲她” → 把他人动机当成事实
- “厨房里的气氛尴尬极了” → 概括气氛
- “两人的关系似乎又回到了原点” → 宣布关系变化
- “像两只刺猬小心翼翼地保持距离” → 俗套比喻

【4. 行动骨架】
水壶沸腾声持续 → 夏知判断必须做点什么打破僵局 → 走向灶台拿起空碗 → 韩川侧身让出空间
韩川继续洗菜，水声盖过沸腾声 → 夏知判断对方不打算主动开口 → 关火、盛面、摆碗 → 韩川把洗好的菜放在案板上
两只碗并排放在灶台上 → 夏知判断这是默许共进晚餐 → 从筷笼抽出两双筷子 → 韩川擦干手，端起其中一碗

【5. 停止保护】
第一个满足stop_state的事实：两人各自端着碗坐下
最易追加的多余内容：
- “韩川终于开口说‘吃饭吧’” → 再次确认已发生的动作
- 夏知坐下后看向韩川，两人对视 → 对视余韵
- “厨房里只剩下筷子碰撞碗沿的声音” → 环境收尾

<story>
水壶的哨声尖锐到刺耳时，夏知还在厨房门口站着。

她看见韩川的手机屏幕亮了一下。屏幕朝上，搁在折叠桌的桌角，离那两只倒扣的碗不远。亮起来的是微信的聊天界面，头像她认得，是她自己的。昨晚发的消息，绿色的气泡还挂在对话框最下方，前面没有红色的未读标记消失——说明他刚才还没点开。

韩川的手从水龙头下抽回来，湿的，在围裙上擦了一下。他低头看了一眼屏幕。夏知看见他的拇指悬在屏幕上方，停了一秒，也许两秒。然后他把手机翻了过去，屏幕贴着桌面，手机壳朝上。黑色的硅胶壳，边角磨得发白。

他转身走到水槽前，拧开水龙头，开始洗那把已经洗过一遍的小白菜。

水壶还在叫。

夏知走进厨房。

她没有去看那只扣着的手机。灶台上的水壶嘴喷出白汽，壶盖被蒸汽顶得轻微跳动。她伸手关了火。哨声断掉之后，厨房里只剩下水龙头的水声和韩川搓菜叶时塑料沥水篮在瓷水槽里晃动的声响。

那只空碗还倒扣在桌上。另一只也是倒扣的。两只碗，两双筷子还插在筷笼里。韩川背对着她，肩膀的轮廓在灰色T恤下绷着，洗菜的动作很规律，像在数叶片。

夏知拿起其中一只碗，翻过来，放在灶台边。碗底在瓷砖台面上磕出一声轻响。韩川的手停了一下，然后继续洗菜。水声没停。

她打开电饭煲。蒸汽扑上来，米香混着水汽。饭勺插在饭里，手柄朝外。她盛了半碗，压了压，又加了一勺，压平。碗沿上沾了一粒米，她用拇指抹掉。

韩川把沥水篮端到案板上。小白菜还在滴水，案板上积了一小摊水迹。他从刀架上抽出菜刀，刀刃在灯光下闪了一下，然后落下去，菜叶被切成宽条，声音规律，砧板笃笃响。

夏知把盛好饭的碗放在桌上，拿起第二只碗。她盛饭的时候，韩川拧开了灶台另一头的炒锅。油倒进去，晃了晃锅底，油纹散开。他把切好的菜推进锅里，刺啦一声，白汽腾起来，蒜味跟着炸开。

夏知把第二碗饭也放在桌上。两只碗并排，碗口冒着热气。她从筷笼里抽出两双筷子，一双横在左边碗上，一双横在右边碗上。

韩川翻炒了几下，关了火。他端起炒锅，把菜拨进一只白瓷盘里。菜叶油亮，蒜末焦黄。他把盘子放在两只碗中间，然后转身去洗锅。

水龙头又响了。

夏知在桌前站了一会儿。两只碗，两双筷子，一盘菜。水槽那边，韩川把锅扣在沥水架上，擦了擦手。他转过身，看见桌上的碗筷，动作顿了一下。

夏知拉开椅子，坐下。椅子腿在瓷砖地面上刮出一声短促的声响。

韩川走过来，在另一只碗前面坐下。椅子被拉开，同样的刮地声，比他平时要轻。

他拿起筷子。夏知也拿起筷子。

两个人谁都没说话。筷子伸向同一盘菜，韩川的筷子先到，夹走了一片菜叶。夏知等他收回筷子，才伸过去，夹起一片。菜叶有点烫，她吹了一下。韩川扒了一口饭。

厨房里只剩下咀嚼声和筷子偶尔碰到盘沿的声响。

夏知吃得很慢。她能感觉到韩川的目光有时候会落在她拿筷子的手上，但每次她抬眼，他都在看碗里的饭。

那只扣着的手机还搁在桌角。屏幕朝下，黑色的壳子吸着从窗户照进来的傍晚光线。没有人去翻它。

韩川先吃完了。他把碗放在桌上，筷子横在碗口。夏知碗里还剩小半碗饭。他站起来，端起空碗和盘子，放进水槽。然后他拿起手机，屏幕翻过来的时候，夏知看见他的拇指在电源键上按了一下，屏幕黑了。

他把手机揣进裤兜，走回水槽前，拧开水龙头。

夏知把最后一口饭咽下去。她站起来，端起自己的碗，走到水槽边。韩川正在挤洗洁精，泡沫从他指缝里溢出来。她把碗放在水槽沿上，离他的手只有十几厘米。

韩川没抬头，但他往旁边挪了半步，给她让出位置。

夏知从挂钩上取下擦桌布，转身去擦桌子。

</story>
